## Media Uniqueness Service

Сервис уникализации видео и изображений, доступный через HTTP‑API.

### Возможности

- Приём файлов по URL через endpoint API.
- Уникализация **видео** с помощью ffmpeg:
  - лёгкие изменения частоты кадров (FPS),
  - корректировка контраста и насыщенности,
  - лёгкий поворот и перекодирование,
  - трекинг прогресса обработки.
- Уникализация **изображений** через Pillow:
  - лёгкое изменение яркости/контраста,
  - размытие, добавление шума,
  - сохранение результата в отдельную папку.
- Асинхронная обработка и **отслеживание статуса задач** через отдельный endpoint.
- Автоочистка старых загруженных и обработанных файлов.

### Основные компоненты

- `main.py` — класс `VID_IMG_EDIT` с логикой обработки и очистки файлов.
- `api.py` — Flask‑приложение с маршрутами:
  - `POST /upload` — загрузка файла по URL и постановка в очередь обработки,
  - `GET /status/<task_id>` — получение статуса задачи,
  - `GET /download/<task_id>` — скачивание готового результата.
- `config_reader.py`, `config.ini` — чтение настроек параметров уникализации (fps, контраст, насыщенность и т.д.).

### Быстрый старт

```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

set FLASK_APP=api.py  # или через .env/переменные окружения
flask run --host=0.0.0.0 --port=8000
```

Убедитесь, что `ffmpeg` и `ffprobe` установлены и доступны в `PATH`.
